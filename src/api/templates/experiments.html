Быстрые команды
make train — обучение без MLflow
make train-mlflow — обучение + метрики и графики (ROC, PR, importance) в MLflow Artifacts → plots/
make drift-check CURRENT=data/incoming/batch.csv — проверка дрейфа
make retrain — переобучение через pipeline script
make docker-up — запустить полный стек (API + MLflow + Prometheus + Grafana)
make minikube-deploy — деплой в Kubernetes (minikube)
